Apps Retired Link to heading

  • YoutubeDL-Material: Worked well but slowly abandoned due to lack of updates
  • lidarr: Music Collection Manager, just didnt use this functionality
  • Bitwarden On-Prem: Migrated to Bitwarden Cloud, didnt see the ROI in self-hosting
  • Pi-Hole: worked well for add blocking just got frustating with some google search results for family. Ultimately i wanted more control so moved to Technitium
  • GitLab: Waaay overkill for what i needed in a git hosting solution. Used a ton of memory as well.
  • Wallabag: didn’t love the interface may come back and try it again at some point
  • Portainer: used for a long time but i don’t like how it hit the stack file locations and didnt track changes as well. Moved to Komodo.
